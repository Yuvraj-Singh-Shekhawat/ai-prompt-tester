## 🧠 Project Overview

This project is an interactive web-based tool designed to simulate real-world **LLM (Large Language Model) evaluation workflows**.

It allows users to:

* Test prompts
* Analyze prompt quality
* Evaluate AI-generated responses

This reflects the core responsibilities required for a Software Engineer role at Ethara.AI.

## ✨ Key Features

### 🔹 Prompt Testing

* Enter custom prompts
* Simulated AI-generated responses

### 🔹 Auto Quality Evaluation

* Prompt Length Analysis
* Clarity Scoring
* Specificity Detection (AI keywords)

### 🔹 Prompt Library

* Save prompts using LocalStorage
* Reload and reuse previous prompts

### 🔹 Interactive UI

* Clean and responsive design
* Keyboard shortcuts (Ctrl+Enter, Ctrl+S)

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API
